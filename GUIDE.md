## 🔧 フェーズA: 3Dプリント

### スライサー設定（Cura等）
| パラメータ | 設定値 |
|-----------|--------|
| ファイル | `mold_cat_clamshell_A.stl` ＋ `mold_cat_clamshell_B.stl` |
| フィラメント | PLA |
| レイヤー高さ | 0.2mm |
| インフィル | 20〜30% |
| 壁の厚さ | 1.2mm（3周） |
| サポート | **なし** |
| ブリム | **あり**（反り防止） |
| 印刷温度 | 200〜210℃ |
| ベッド温度 | 60℃ |

💡 **配置のコツ**: 各ハーフは**フランジ面を上に向けて**プリントすると、ピン/穴の精度が出やすく、フランジ平面も綺麗になります（ベッド側は曲面のないキャビティ背面側）。

### 印刷後
1. ブリムをカッターで除去
2. 内面（キャビティ）をサンドペーパーで研磨
   - **#400** → 積層跡を除去
   - **#800** → 滑らかに仕上げ
3. **フランジ平面**は#400で軽く撫でて平らにする（接合精度UP）
4. ハーフAのピンとハーフBの穴を実際に合わせて、スッと嵌まることを確認

## 🧪 フェーズD: バスボム制作

### D-1. 粉末材料を混ぜる
```
ボウルに入れる順番:
  ① 重曹         200g
  ② クエン酸      100g
  ③ コーンスターチ  50g
  ④ 竹炭パウダー   小さじ1〜2（少量から！）
```
- ゴム手袋を着用！（竹炭は手につくと落ちにくい）
- フォークでダマをつぶしながらよく混ぜる（2〜3分）

### D-2. 液体材料を加える
```
  ⑤ ココナッツオイル   大さじ1〜2（固形の場合はレンジで溶かす）
  ⑥ ラベンダーEO      10〜15滴
```
- よく混ぜる

### D-3. 湿度を調整する
```
  ⑦ 無水エタノールをスプレーボトルに入れる
    → 混合物に「シュッ、シュッ」と1〜2回ずつ吹きかけながら混ぜる
```
- 🎯 **完成の目安**: ギュッと握ると固まり、軽く押すと崩れる状態
  - 「湿った砂」のような質感
  - ポロポロすぎる → もう1プッシュ
  - ベチャベチャ → 入れすぎ！重曹を少し追加

- ⚠️ **絶対に水を使わない！** 水だとシュワシュワ反応が始まってしまう

### D-4. クラムシェル型に詰める

```
   ハーフA（ピン側）         ハーフB（穴側）
   ┌─────────────────┐      ┌─────────────────┐
   │  ① 内面に離型処理 │      │  ① 内面に離型処理 │   ← サラダ油を薄塗り or ラップ
   │                  │      │                  │
   │  ② 山盛りに材料を │      │  ② 山盛りに材料を │   ← キャビティが少し
   │     詰める ↑↑↑   │      │     詰める ↑↑↑   │     はみ出るくらい
   │                  │      │                  │
   │  ③ ギュッと押し  │      │  ③ ギュッと押し  │
   │     固める       │      │     固める       │
   └─────────────────┘      └─────────────────┘
                  ↓ 合わせる ↓
            ┌─────────────────┐
            │   両ハーフを合体  │   ← ピンを穴に差し込みながら
            │     ピン → 穴    │     ぐっと押し付ける
            │   (はみ出した分が │
            │    つぶれて密着)  │
            └─────────────────┘
                  ↓ 圧着 ↓
            ┌─────────────────┐
            │  ④ 輪ゴムで       │   ← 縦・横にX字にかける
            │     しっかり固定  │     (フランジ部分で押さえる)
            └─────────────────┘
```

#### 詳細手順:
1. **離型処理**: 各ハーフのキャビティ内面に、サラダ油 or ココナッツオイルを薄く塗る（指 or キッチンペーパーで）。ラップを敷くより推奨（クラムシェルだとラップは逆にしわが入りやすい）
2. **両ハーフに材料を詰める**: 半量ずつ、各ハーフのキャビティに **少し山盛り** に詰める（キャビティ容量の少し多め＝合体時に圧縮される分の余裕）
3. **ギュッと押し固める**: 各ハーフ内で指の腹でしっかり圧縮
4. **ハーフA（ピン側）の上にハーフBをかぶせる**: ピンと穴を合わせながらまっすぐ降ろし、ぐっと押し付ける。フランジ同士がぴったり合うまで
5. **輪ゴムで圧着**: フランジ部分にX字（縦＋横）に輪ゴムをかけて固定。緩いと隙間ができるので、しっかり締める

## ⏳ フェーズE: 乾燥

1. クラムシェルを輪ゴムで圧着したまま、クッキングシートの上に置く
2. **風通しの良い場所**で **24〜48時間** 放置
3. ⚠️ **触らない！** 乾燥中は非常にもろい
4. ⚠️ 湿気の多い場所（風呂場・キッチン）は避ける
5. 夏場は24時間、冬場は48時間が目安

## 🎁 フェーズF: 仕上げ・ラッピング

### 型から取り出す（クラムシェル開封）
1. 輪ゴムをそっと外す
2. 型のフランジ部分を両手で持ち、**左右にゆっくり開く**（パカッと貝殻のように）
3. バスボムは通常どちらか片方のハーフに残るので、ハーフをひっくり返して**手のひらに優しく落とす**
4. もし張り付いて落ちない場合：ハーフの背面（キャビティの裏側）を反対の手で軽くトントンと叩く
5. ⚠️ 力を入れすぎない（耳や触手が折れやすい）

### 仕上げ
- パーティング面（合わせ目）に薄くバリが出ることがあるので、指で軽く撫でて整える
- 角が欠けた場合は、余った混合物を指で押し付けて補修

### ラッピング
1. バスボムをラップでぴったり包む（湿気防止）
2. 透明袋に入れてリボンで結ぶ
3. 使い方カードを添える（下記参照）

## ⚠️ トラブルシューティング

| 問題 | 原因 | 対策 |
|------|------|------|
| 固まらない・ポロポロ | エタノール不足 | もう1〜2プッシュ追加 |
| シュワシュワし始めた | 水分が入った | 素早く型に詰める。次回はエタノールのみ使用 |
| 型から出ない | 離型剤を塗り忘れた / 圧縮しすぎ | 油を必ず薄塗り。冷凍庫30分→再挑戦 |
| クラムシェルが閉じない | 材料を詰めすぎ | 詰める量を減らす（少し山盛り＝合体で圧縮される程度） |
| 合わせ目に隙間ができる | 輪ゴムが緩い | より強い輪ゴムでX字にしっかり固定 |
| 耳が折れた | 薄い部分は壊れやすい | 余った材料で指で補修。次回は少し厚めに |
| 乾燥中にひび割れ | 乾燥が速すぎる / 厚みが不均一 | 乾燥場所を変える。型を密閉した状態でゆっくり乾燥 |
| お湯が真っ黒 | 竹炭パウダー入れすぎ | 次回は小さじ1/2から調整 |
| 浴槽に色が残った | 竹炭 or 食用色素 | メラミンスポンジで落ちる。竹炭の方が残りにくい |

## 💡 成功のコツ

1. **初回は練習**: 余った材料で丸いミニバスボム（手で丸める）を1個作って感覚をつかむ
2. **竹炭は少量から**: 小さじ1/2で十分黒くなる。多すぎるとお湯が真っ黒に
3. **離型剤は薄く**: 油を塗りすぎるとバスボムの表面がベタつく
4. **押し固める力加減**: 思ったより強めにギュッと。弱いと崩れる
5. **エタノールは少しずつ**: 入れすぎより入れなさすぎの方がマシ
6. **保管は密閉**: 完成品は湿気を避け、1〜2週間以内にプレゼントする
