## 🔧 フェーズA: 3Dプリント

### スライサー設定（Cura等）
| パラメータ | 設定値 |
|-----------|--------|
| ファイル | `mold_cat_container.stl` |
| フィラメント | PLA |
| レイヤー高さ | 0.2mm |
| インフィル | 20〜30% |
| 壁の厚さ | 1.2mm（3周） |
| サポート | **なし** |
| ブリム | **あり**（反り防止） |
| 印刷温度 | 200〜210℃ |
| ベッド温度 | 60℃ |

### 印刷後
1. ブリムをカッターで除去
2. 内面をサンドペーパーで研磨
   - **#400** → 積層跡を除去
   - **#800** → 滑らかに仕上げ
3. 内面を指で触ってツルツルになっていればOK

## 📷 フェーズD: 写真の防水加工

### 手順
1. 写真を **85mm × 55mm 以内** に印刷（コンビニプリントOK）
   - ⚠️ 型の底面サイズは86×92mm。写真の長辺を92mm方向に合わせる
   - ⚠️ インクジェット印刷は水に弱いので**必ず**ラミネート加工する
2. 手貼りラミネートフィルムで写真を挟む
   - **周囲に5mm以上の余白**を残して密着させる
   - 端を爪や定規でしっかり押さえて密封
3. はみ出たフィルムを写真の形に沿ってカット

### ⚠️ 防水テスト（必須！）
- ラミネートした写真を**水に5分間浸ける**
- 取り出して水気を拭き、写真が濡れていないか確認
- 濡れていたら → ラミネートをやり直す

## 🧪 フェーズE: バスボム制作

### E-1. 粉末材料を混ぜる
```
ボウルに入れる順番:
  ① 重曹         200g
  ② クエン酸      100g
  ③ コーンスターチ  50g
  ④ 竹炭パウダー   小さじ1〜2（少量から！）
```
- ゴム手袋を着用！（竹炭は手につくと落ちにくい）
- フォークでダマをつぶしながらよく混ぜる（2〜3分）

### E-2. 液体材料を加える
```
  ⑤ ココナッツオイル   大さじ1〜2（固形の場合はレンジで溶かす）
  ⑥ ラベンダーEO      10〜15滴
```
- よく混ぜる

### E-3. 湿度を調整する
```
  ⑦ 無水エタノールをスプレーボトルに入れる
    → 混合物に「シュッ、シュッ」と1〜2回ずつ吹きかけながら混ぜる
```
- 🎯 **完成の目安**: ギュッと握ると固まり、軽く押すと崩れる状態
  - 「湿った砂」のような質感
  - ポロポロすぎる → もう1プッシュ
  - ベチャベチャ → 入れすぎ！重曹を少し追加

- ⚠️ **絶対に水を使わない！** 水だとシュワシュワ反応が始まってしまう

### E-4. 型に詰める

```
    ┌───型（猫型容器）──┐
    │                    │
    │  ① ラップを敷く    │  ← ラップが壁と底に密着するように
    │                    │
    │  ② 材料を半分詰める │  ← ギュッと押し固める
    │     ↓↓↓↓↓↓        │
    │  ③ 写真を置く 📷   │  ← 中央に平らに
    │                    │
    │  ④ 残りを全部詰める │  ← ギュッと押し固める
    │     ↓↓↓↓↓↓        │
    │  ⑤ ヘラで平らにする │  ← 上面をスーッと整える
    └────────────────────┘
```

#### 詳細手順:
1. **ラップを敷く**: 型の内面に食品用ラップをぴったり敷く
   - しわを最小限に（しわの跡がバスボムに残るため）
   - ラップの端を型の外側に垂らしておく（後で引き上げるため）
2. **材料を半分詰める**: 混合物を型の半分の深さまで入れ、指でギュッと押し固める
3. **写真を置く**: ラミネート済み写真を中央に平らに置く
   - 写真の長辺(85mm)を型の長い方向(92mm)に合わせる
4. **残りを詰める**: 残りの混合物を上から入れ、しっかり押し固める
   - 写真が完全に隠れていることを確認
5. **上面を平らにする**: ヘラ or 定規で上面をスーッと平らに整える

## ⏳ フェーズF: 乾燥

1. 型に入れたまま、クッキングシートの上に置く
2. **風通しの良い場所**で **24〜48時間** 放置
3. ⚠️ **触らない！** 乾燥中は非常にもろい
4. ⚠️ 湿気の多い場所（風呂場・キッチン）は避ける
5. 夏場は24時間、冬場は48時間が目安

## 🎁 フェーズG: 仕上げ・ラッピング

### 型から外す
1. 型をひっくり返す
2. ラップの端を持って**ゆっくり引き上げる**
3. バスボムが出てきたら、ラップを静かに剥がす
4. ⚠️ 力を入れすぎない（耳や触手が折れやすい）

### 仕上げ
- 表面のラップのしわ跡を指で軽くなでて整える
- 角が欠けた場合は、余った混合物を指で押し付けて補修

### ラッピング
1. バスボムをラップでぴったり包む（湿気防止）
2. 透明袋に入れてリボンで結ぶ
3. 使い方カードを添える（下記参照）

## ⚠️ トラブルシューティング

| 問題 | 原因 | 対策 |
|------|------|------|
| 固まらない・ポロポロ | エタノール不足 | もう1〜2プッシュ追加 |
| シュワシュワし始めた | 水分が入った | 素早く型に詰める。次回はエタノールのみ使用 |
| 型から出ない | ラップを敷き忘れた / 抜き勾配不足 | ラップを必ず敷く。冷凍庫30分→再挑戦 |
| 耳が折れた | 薄い部分は壊れやすい | 余った材料で指で補修。次回は少し厚めに |
| 乾燥中にひび割れ | 乾燥が速すぎる / 厚みが不均一 | 乾燥場所を変える。ラップで軽く覆う |
| 写真が濡れた | ラミネートの密封不良 | 端をもう一度押し付ける or やり直し |
| お湯が真っ黒 | 竹炭パウダー入れすぎ | 次回は小さじ1/2から調整 |
| 浴槽に色が残った | 竹炭 or 食用色素 | メラミンスポンジで落ちる。竹炭の方が残りにくい |

## 💡 成功のコツ

1. **初回は練習**: 余った材料で丸いミニバスボム（手で丸める）を1個作って感覚をつかむ
2. **竹炭は少量から**: 小さじ1/2で十分黒くなる。多すぎるとお湯が真っ黒に
3. **ラップは最重要**: 型の離型と衛生面の両方を守る
4. **押し固める力加減**: 思ったより強めにギュッと。弱いと崩れる
5. **エタノールは少しずつ**: 入れすぎより入れなさすぎの方がマシ
6. **保管は密閉**: 完成品は湿気を避け、1〜2週間以内にプレゼントする
